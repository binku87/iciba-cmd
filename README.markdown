# iciba-cmd #

iciba-cmd is a command line tools which used to search words and play sound .


## How to setup ##

1.Download mpg123

  sudo apt-get install mpg123

2.make iciba executable 
    
  sudo chomd +x iciba

3.copy to bin directory,such as 'sudo cp iciba /usr/bin/'

4.Usage:

  ~/Desktop$ iciba word
